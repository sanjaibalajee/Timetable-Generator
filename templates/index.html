An Automatic Timetable Generation System

An automatic Timetable generation system is a tool developed using Python to generate efficient timetables for universities based on set of constraints and inputs. The online timetable Generator extinguishes paper-dependency by proving automated and digitized solutions for the problem.


Student Timetable

Generates a well-constructed timetable which establishes a natural rhythm and routine, which can be comforting to students.


Faculty Timetable

Generates a confusion free timetable for every induvidual teacher. The timetable notifies the teachers of which subject to expect the next day.
